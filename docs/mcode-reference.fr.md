# Référence M‑code (résumé)

Version abrégée en français — voir `docs/mcode-reference.md` (anglais) pour la liste complète.

Exemples fréquents :
- `M80` / `M81` — PSU on/off
- `M106` / `M107` — ventilateur (P/S)
- `M150` — LED/RGB
- `M191` — attendre la température de chambre
- `M240` — déclencheur appareil photo (selon firmware)
- `M928` / `M29` — journal sur carte SD
