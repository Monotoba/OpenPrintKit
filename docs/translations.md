# Translations

This project welcomes translations of the User Manual and key docs. Suggested workflow:

1) Copy `docs/user-manual.md` to `docs/user-manual.<lang>.md` (e.g., `user-manual.es.md`).
2) Translate headings and content; keep command examples as‑is.
3) Add a language note at the top and link back to the English original.
4) Submit a PR with the translated file and an index entry in `docs/index.md`.
