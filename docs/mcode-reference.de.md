# M‑Code Referenz (Kurzfassung)

Kurzfassung — vollständige Liste siehe `docs/mcode-reference.md` (Englisch).

Häufige Codes:
- `M80` / `M81` — PSU an/aus
- `M106` / `M107` — Lüfter (P/S)
- `M150` — LED/RGB
- `M191` — Warte Kammer‑Temperatur
- `M240` — Kameraauslöser (firmwareabhängig)
- `M928` / `M29` — SD‑Logging
