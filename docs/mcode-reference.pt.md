# Referência de M‑code (resumo)

Resumo — lista completa em `docs/mcode-reference.md` (inglês).

Códigos comuns:
- `M80` / `M81` — PSU ligar/desligar
- `M106` / `M107` — ventilador (P/S)
- `M150` — LED/RGB
- `M191` — aguardar temperatura da câmara
- `M240` — disparo da câmera (depende do firmware)
- `M928` / `M29` — log no SD
